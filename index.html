Då man skapar en layout med grid används CSS-egenskaperna grid-template-columns, grid-template-rows och grid-template-areas. Beskriv vad dessa används till.

Med hjälp av grid skapas webbsidans layout som man kan likna celler i form av en tabell med rader och kolumner. Cellens innehåll bestäms i grid-template-areas, cellens bredd och höjd i respektive grid-template-columns och grid-template-rows.

Vad är en media query och vad används den till?

Media query används för att anpassa webbsidans innehåll till skärmar av olika storlekar. Med hjälp av media query kan man omstrukturera/anpassa delar av webbsidans innehåll så att den passar t.ex. mobil- eller surfplattaskärm. Det görs med hjälp av css kod.
